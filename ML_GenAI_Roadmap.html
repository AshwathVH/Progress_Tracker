📌 Getting Started & Best Practices
1️⃣ DAILY WORKFLOW
• Morning: Check "Today's Goal" - It's automatically calculated from the daily goals section
• Start: Click the "Start" button on the Study Session Timer
• Work: Read concepts first (10-15 min), then code along with tutorials (usually 60-90 min)
• Practice: Solve practice problems or exercises (remaining time)
• Log: Check off completed tasks as you finish them
• Evening: Review what you learned and plan tomorrow
2️⃣ TIME TRACKING
• Each task has a time estimate (e.g., 1.5 hrs) based on realistic study pace
• Use the Study Session Timer to automatically track time spent
• "Time Spent" box shows: Manual task times + Timer elapsed time
• Goal: Try to meet or exceed your daily time goal (aim for consistency)
• Maximum: 6-7 hours/day recommended to avoid burnout
3️⃣ CONCEPTS & MASTERY
• Each daily block has a "Key Concepts to Master" section
• ✓ marks show concepts specific to that day
• For each concept: Read theory → Code examples → Build small project
• Don't move to next day until 80% confident in current day's concepts
• Review concepts from previous days weekly
4️⃣ TRACKING YOUR PROGRESS
• Progress Bar: Shows overall % of roadmap completed
• Weekly Summary: Tasks done this week + hours invested
• Study Streak: Days you've studied consistently (counts days with ≥1 task completed)
• Day Progress: Green bar under each day shows how many tasks in that day are done
• Data automatically saves to browser storage (persists between sessions)
5️⃣ SAMPLE DAILY SCHEDULE (6-hour day)
09:00 - 09:30 | ☕ Coffee + Review yesterday
09:30 - 10:45 | 📚 Theory & Concepts (75 min)
10:45 - 11:00 | ☕ Break
11:00 - 13:00 | 💻 Hands-on Coding (120 min)
13:00 - 14:00 | 🍽️ Lunch Break
14:00 - 15:30 | 🧠 Practice Problems (90 min)
15:30 - 16:00 | 📝 Notes & Summary
Total: 6 hours
6️⃣ TIPS FOR SUCCESS
✅ Consistency over intensity: 5-6 hours daily is better than 12 hours once a week
✅ Code along: Never just read code - type it out yourself
✅ Keep projects: Save all practice code in GitHub for portfolio
✅ Take breaks: 5 min per hour helps memory consolidation
✅ Teach others: Explain concepts to friends or online communities
✅ Stay flexible: If a topic takes longer, spend extra days on it
✅ Weekly review: Every Sunday, review the week's major concepts
⚠️ COMMON MISTAKES TO AVOID
❌ Skipping fundamentals to jump to ML/AI
❌ Not doing hands-on projects regularly
❌ Forgetting to save work to GitHub
❌ Trying to memorize instead of understanding
❌ Not taking breaks (causes burnout)
❌ Inconsistent study schedule (one day off breaks streak)
